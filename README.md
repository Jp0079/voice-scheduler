Instructions to test the agent : 
1) Click on the above link and click on "Talk to VAPI".
2) The agent will ask you the purpose of the call and you can say -> "I want to book an appointment."
3) Then the agent will ask for your Full Name and your exact date and time, which you need to share, for example : "Tuesday, 06th December at 11:00 AM IST"
4) If the given slot is available, then the agent will once again confirm the details with you, otherwise it will give you the list of available slots.
5) Finally, you event would be booked.
